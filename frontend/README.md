# Stock Metrics UI

React + Vite skeleton with Keycloak authentication, Redux Toolkit, Bootstrap, Chart.js, and WebSocket price streaming stubs.

## Run
```bash
npm install
npm start
```

Configure environment (optional):
- `VITE_KEYCLOAK_URL` (default `http://localhost:8080`)
- `VITE_KEYCLOAK_REALM` (default `stock-metrics`)
- `VITE_KEYCLOAK_CLIENT` (default `stock-metrics-ui`)
- `VITE_STREAM_URL` (default `ws://localhost:8082`)
